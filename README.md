# Web Templates

Kolekcja szablonów stron internetowych.

## Projekty

### justyna-serafin
Tradycyjna strona wielostronicowa dla ortodonty.
- Technologie: Astro, Tailwind CSS
- Jasny motyw z gradientami
- Oddzielne strony: strona główna, cennik, blog, aktualności

### justyna-serafin-v2
Nowoczesna strona SPA dla ortodonty.
- Technologie: Astro, Tailwind CSS v4
- Przełączanie motywów jasny/ciemny (domyślnie jasny)
- Nawigacja kategoriami bez przeładowania
- Design inspirowany webflow
- Typografia: Inconsolata + Inter

## Uruchomienie projektów

```bash
cd justyna-serafin
pnpm install
pnpm dev
```

lub

```bash
cd justyna-serafin-v2
pnpm install
pnpm dev
```
